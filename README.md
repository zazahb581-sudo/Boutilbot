# Boutilbot

Boutilbot is a virtual crypto exchange Telegram bot.

## Features

- Buy USDT
- Buy BTC
- Instant order creation
- Order ID system
- 24/7 customer support
- Bankily payment support

## How It Works

1. Open the Telegram bot
2. Create an order
3. Send payment via Bankily
4. Send payment proof to support
5. Order is confirmed manually
